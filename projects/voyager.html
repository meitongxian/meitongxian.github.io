VOYAGER

Type, Code
Spring 2025–

Advised by Cyrus Highsmith for Grad Type Design @ RISD.

A soft, yet steady text serif, initially designed for Antoine de Saint-Exupéry’s novella, The Little Prince.

Voyager is low-contrast and highly legible in smaller sizes, but its gentle serifs and ligatures maintain character when used for bigger headings (or, the larger text of a children’s book).

This typeface is a work in progress — expanded character set, italics, and further documentation coming soon. Made using Glyphs.
